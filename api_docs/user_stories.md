**<h1>User Stories</h1>**

# **User**

## **Sign Up**

As an unregistered and unauthorized user, I want to be able to sign up for the website via a sign-up form. When I'm on the /signup page:

*  I would like to enter my email, username, and password on a clearly laid out form.

* I would like the website to log me in upon successful completion of the sign-up form, so I can seamlessly access the site’s functionality.

* When I enter invalid data on the sign-up form, I would like the website to inform me of the validations I failed to pass and repopulate the form with my valid entries (except my password).   So that I can try again without needing to refill the form for valid data entries.

## **Log in**

As a registered and unauthorized user, I want to be able to log in to the website via a log-in form. When I'm on the /login page:

*  I would like to enter my email and password on a clearly laid out form.

*  I would like the website to log me in upon successful completion of the login form, so I can access the site's functionality.

*  When I enter invalid data on the login form, I would like the website to inform me of the validations I failed to pass and repopulate the form with my valid entries (except my password).   So that I can try again without needing to refill forms for valid entries.

## **Demo User**

 As an unregistered and unauthorized user, I want a demo button on both the /signup and /login pages to visit the site as a guest without signing up or logging in. When I'm on either the /signup or /login pages:

*  I can click a "Demo User" button to log me in as a guest user.   So that I can test the site's features and functionality without needing to enter credentials.

## **Log Out**

As a logged-in user, I want to log out via an easy-to-find "Log Out" button on the navigation bar.
While on any page of the site:

* I can log out of my account and be redirected to a page displaying recent products or tracks.
  So that I can easily log out and keep my information secure.

# **POSTS**

## **View Posts**

* As a user, I want to browse all posts so I can explore available tattoos and designs.

* As a user, I want posts to display details like images, price, and description so I can understand what’s being offered.

## **Create Posts**

* As a logged-in user, I want to create posts with details like name, description, price, and images so I can share my work with others.

## **Update Posts**

* As a logged-in user, I want to edit my posts to update details like price, description, or images so they stay accurate and appealing.

* I can click "Edit" to modify details such as price, description, images, or booking status. For example, I can edit whether the flash is "available" for booking or not, depending on if I want to repeat designs or not.

## **Delete Posts**

* As a logged-in user, I want to be able to delete my product listings.

* I can click "Delete" to remove a post I no longer want to showcase.

<br></br>

# **COMMENTS**

## **View Comments on Post**

As a logged-in or logged-out user, I want to read comments on posts.
When I'm on the /post/:id page:

* I can see comments left by other users so that I can make informed booking decisions based on others’ feedback. As an artist, I want to view all comments on a post so I can see feedback from others.

## **Create a Comment**

As a logged-in user, I want to be able to leave a comment on artists' posts.

* I can write and submit a comment on a tattoo post so I can share my thoughts or ask questions.

## **Update a Comment**

As a logged-in user, I want to be able to edit my comment.

* I can click "Edit" to update the content of my comment so I can correct mistakes or update my feedback.

## **Delete a Comment**

As a logged-in user, I want to delete my comments so I can remove feedback I no longer want to share.

* I can click "Delete" to remove a comment I no longer want to share so that I can manage my contributions to the platform.

# **LIKES/FAVORITES**

## **View Likes**

* As a user, I want to see the number of likes on a post so I can gauge its popularity.

* As a logged-in user, I want to view all the posts I’ve liked on a Favorites page so I can easily find them again.

## **Add a Like**

* As a logged-in user, I want to like a post so I can express my appreciation and save it for later.

## **Delete a Like**

* As a logged-in user, I want to remove my like from a post so I can adjust my preferences.

<br></br>

# **BOOKINGS**

## **View Booking Requests**

* As a tattoo artist, I want to see all the booking requests I’ve received so I can manage client inquiries.

## **Create Booking**

* As a client, I want to submit a booking request for a specific tattoo post so I can schedule an appointment.

## **Update Booking**

* As a client, I want to edit the date of a booking request so I can adjust my appointment to fit my schedule.

* As a client, I want to be notified of any deposit loss when updating my booking past the artist’s cancellation policy.

## **Delete Booking**

* As a client, I want to cancel my booking request so I can withdraw my interest or if my schedule/finances prevent me from making the tattoo appointment.

* As a client, I want to understand the cancellation policy and deposit implications when deleting a booking.

# **Bonus Features**

## **Search Functionality**

* As a user, I want to search for posts using keywords so I can quickly find relevant tattoos or artists.

## **Messaging**

* As a client, I want to message an artist after my booking request is approved so we can discuss details.

* As an artist, I want to message a client about their approved booking request so we can coordinate the appointment.
